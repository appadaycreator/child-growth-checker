  - 「成長曲線を確認する」ボタン押下後、結果見出しがビューポート上部に配置される（block: 'start' に変更）

  - 破線の太さを 1px → 1.5px に変更
  - より見やすくなる
- **P2: パーセンタイルバーのマーカーに数値ラベルを追加:**
  - マーカー直上に「〇〇%」のラベルを表示
  - マーカー位置と実際の数値が対応しやすくなる
- **P2: 目標設定セクションをデフォルト展開:**
  - ページロード直後、目標設定セクションが表示される状態に変更（従来は折りたたみ）
  - セクションの背景色を黄系（#fef3c7）に変更して視認性向上
  - 初回ユーザーが目標設定機能の存在を認識しやすくなる

## v2.1.0 更新内容（2026-06-10）
**P1・P2改善（M6・M10機能追加）:**
- **P1: 目標設定・達成度トラッキング機能（M6）を実装:**
  - 身長・体重の目標値を設定可能
  - 達成度をプログレスバーで視覚化（0〜100%）
  - 前回測定値との差分を自動計算・表示
  - 目標値をlocalStorageに永続化
  - 「目標を設定する」セクション（収納可能）
- **P1: 結果の印刷・CSVエクスポート機能（M10）を実装:**
  - 「🖨️ 印刷」ボタン → window.print()で印刷可能
  - 「📥 CSV」ボタン → 測定値・パーセンタイル・カウプ指数をCSVダウンロード
  - 既存の @media print CSS を活用（ヘッダー・フッター・広告は非表示）
  - CSVには日付、年齢、性別、身長、体重、パーセンタイル、カウプ指数を含む
  - 前回測定値をキー「cgc_h_prev」「cgc_w_prev」で管理

子どもの年齢・身長・体重を入力して成長曲線のパーセンタイルを確認。発育の目安をチェック。登録不要・完全無料でご利用いただけます。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. ページを開き、入力フォームの項目を確認する
2. 必要な情報を入力または選択する
3. 実行ボタンをクリックして結果を取得する
4. 表示された結果・アドバイスを確認する
5. 必要に応じてコピー・SNSシェアで活用する
